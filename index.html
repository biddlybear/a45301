A453 Programming Project

This website has been created to help those students that are struggling a little with the concepts that are required for completion of the A453/01 Programming project (affectionately known as the "Dungeons and Dragons one".)

Full instructions on how to complete the task are given, and some rough guidelines are available to help where required. If you have any questions, tweet me @biddly_bear.
