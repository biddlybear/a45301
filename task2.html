Creating Some Characters.
Do you have the strength and skill for Task 2?

Your task is to create a program that will calculate the characteristics of two characters' attributes. The only two characteristics you need to calculate are Strength and Skill.

Both characters have strength and skill set at 10 by default. A 12-sided dice and a four-sided dice are rolled. You then divide the result of the 12 sided dice by the result of the four sided dice, and round down. This number is then added to the initial value. This will happen four times in total(CharA Strength, CharA Skill, CharB Strength, CharB Skill). Finally, save these results into a file.

Please note that for the purposes of simplicity, this website does not store the values that are generated.
