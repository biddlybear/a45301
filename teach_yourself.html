A Teacher Helping Teachers.

For teachers that are getting used to programming in general, or more specifically Python, here are a few Python-related links to help you along. I actually made this list for myself as a reference but am more than happy to share.

A lot of these resources have helped me in the past, but I'm not 100% sure how up-to-date they all are. If you have any questions or want to chat, tweet me @biddly_bear, or head on over to my blog to read the ramblings of someone that likes Computer Science probably far too much.

First up, the book that I initially learned Python from (NB: some of this won't work due to changes between Python 2.x and Python 3.x, but the general work still stands: How to Think Like a Computer Scientist, or download the ebook.

I would recommend using these two to actually teach and learn from, though, due to their interactive natures: Codecademy and Learning with Python: Interactive Edition 2.0.

Then a couple of books that have some slightly more advanced topics: A Hacking with Python book and An intermediate Python Games Development book from the same publisher.

Just in case you hadn't had enough by now, Google use Python as their main scripted language (alongside C++ on the data processing side and a bit of Java here and there to sew things up). Why not have a read of the style guide regarding how to lay out a program, called the Google Python Guide - advanced reading, sure, but just in case it takes your interest, have a go.
