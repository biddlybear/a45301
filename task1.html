Time to get started.
Let's roll the dice in Task 1.

Your task is to create a program that will roll a 4, 6 or 12 sided dice. The user should be able to input (or choose) what dice should be thrown, and the program should output the dice chosen and score. The user needs to be able to repeat this as many times as required.

Below are a couple of representations of how your final program might look. Don't base your designs on this, it is just an example of how the program should work.
